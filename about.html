-Our Mission-

To find least time consuming path between source and destination

-About-

This project use Dijkstra's Algorithm to find least time consuming path between source and destination using Rail Route and Air Route.

  • Route Finder

    It take source and destination form user and find least time consuming path using available small paths from database.

  • Upload Form

    Form to put more path in the database.
    It takes source, destination , travel time and type of transport and add new path to database.
